# Lindenweg Dashboard

Skandinavisch / Muji-inspiriertes Home Assistant Custom Panel.

- Räume, Szenen, Energie, Wetter, Sicherheit, Kameras, Kalender
- Linen (hell) und Walnut (warmes Dark) Theme
- Vollgrafischer Editor für Entity-Zuordnung — kein YAML nötig
- Lit + TypeScript, einzelnes Bundle (~ unter 100 kB minified)

Nach der Installation einen `panel_custom`-Eintrag in `configuration.yaml` hinzufügen — siehe README.
